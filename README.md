## 💻 Sobre o Projeto

Este projeto consiste no desenvolvimento Front-End da plataforma da Banda Sinfônica Nacional. O objetivo foi transformar os protótipos de alta fidelidade desenhados no Figma em uma aplicação web real, rápida e interativa. Todo o fluxo de trabalho, desde a ideação visual até a componentização e o deploy, foi gerenciado de ponta a ponta.

### ✨ Principais Funcionalidades

* **Design Responsivo:** Interface fluida que se adapta perfeitamente a dispositivos móveis, tablets e desktops.
* **Componentização Moderna:** Arquitetura baseada em componentes reutilizáveis para facilitar a manutenção e escalabilidade do código.
* **Otimização de Performance:** Carregamento de assets otimizado e estruturação semântica.
* **UI/UX Fiel ao Protótipo:** Tradução precisa do design system criado no Figma para o código (Pixel Perfect).

---

## 🛠️ Tecnologias Utilizadas

* **Front-End:** React.js, Vite
* **Estilização:** Tailwind CSS (para um desenvolvimento ágil e padronizado)
* **Design & Prototipação:** Figma
* **Organização e Documentação:** Notion

## 🚀 Como Executar o Projeto Localmente

Siga os passos abaixo para rodar a aplicação no seu ambiente de desenvolvimento:

1. **Clone este repositório**
   ```bash
   git clone [https://github.com/annielarcher/queen-bsn.git](https://github.com/annielarcher/queen-bsn.git)

2. **Acesse a pasta do projeto**
   ```bash
   cd queen-bsn

3. **Instale as dependências**
    ```bash
    npm install

4. **Inicie o servidor de desenvolvimento**
    ```bash
    npm run dev

5. **Acesse no navegador**
   Abra http://localhost:5173 (ou a porta indicada no seu terminal).
